## Setup
1. Frontend:
   - `cd frontend`
   - `npm install`
   - `npm run dev`
2. Backend:
   - `cd backend`
   - `npm install`
   - Create `.env` with DATABASE_URL and CLOUDINARY credentials
   - `npm start`
